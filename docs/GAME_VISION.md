# El Caminante

Juego narrativo cenital de peregrinación cristiana para público adulto. Su identidad evangélica es explícita, pero sus conflictos se expresan mediante observación, conversación, discernimiento y aplicación, sin combate ni puntos de vida.

El primer vertical slice es **El precio del atajo**, situado en el Mercado del Umbral. El jugador reúne indicios, interpreta la oferta de Apatē y decide cómo aplicar lo comprendido. El resultado altera de forma persistente el mercado, las estadísticas narrativas, el diario y el códice.

## Principios

- La verdad se descubre y se practica; no funciona como una respuesta de trivia.
- Las consecuencias son visibles, persistentes y continuables después del encuentro.
- Apatē conserva aspecto humano y permanece en el mercado.
- La Escritura se referencia con cuidado. Este slice cita Hebreos 3:13, Colosenses 2:8 y Mateo 13:22, sin incorporar todavía sus textos.
- Español es el idioma inicial; todo texto visible tiene equivalente inglés.

La segunda etapa añadirá refugio/fogata, memorización progresiva del Salmo 23 y el Valle de Sombra.
